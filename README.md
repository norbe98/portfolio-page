# Personal Portfolio Website

A clean, responsive, and performance-optimized portfolio website built to showcase my projects, skills, and professional journey as a developer.

🔗 **Live Demo:** 

---

## 🚀 Technologies Used

- **Core:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS

---

## ✨ Key Features

- **Dynamic Mouse Glow:** A lightweight, optimized radial gradient effect that dynamically tracks the cursor without lagging the UI.
- **Sticky Sidebar Layout:** A modern, desktop-first design featuring a fixed left column for profile information and a scrollable right column for content sections.
- **Component-Driven Architecture:** Clean separation of concerns with fully modular, reusable React components.
- **Fully Responsive:** Styled using Tailwind's responsive utilities to ensure a flawless experience across mobile, tablet, and desktop screens.
